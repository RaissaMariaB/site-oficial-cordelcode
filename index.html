Cordel Code

O Cordel Code nasceu com a proposta de propagar conhecimento como uma forma de fortalecer e emponderar mulheres. Tornando-as mais preparadas ao mercado de trabalho, onde o grupo discute, faz exercícios propostos por elas mesmas, num ambiente favorável ao desenvolvimento intelectual. Proporcionando assim a qualificação profissional de mulheres de diversas áreas do conhecimento, sendo ligadas a áreas de tecnologias ou não, em desenvolvedoras de Front-End. As nossas desenvolvedoras de Front-End, trabalham com HTML5, CSS, JavaScript, para produzir a interatividade entre a máquina-usuário com os layout’s, fazendo tudo com responsividade. Compartilhando conhecimento ele se propagar, gerando mais profissionais capazes para o mercado de trabalho e aumentando a quantidade de mulheres nesse mercado ainda de predominância masculina.

A essência do conhecimento consiste em aplicá-lo, uma vez possuído.
